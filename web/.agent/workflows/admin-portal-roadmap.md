# Admin Portal Improvement Roadmap

This workflow tracks the implementation of the "All Suggestions" request for the Liberty Meal Prep admin portal.

## Phase 1: Foundation & UX (Current Focus)
- [ ] **Toast Notifications**: Replace `alert()` with `sonner` for better UX.
- [ ] **Search & Filtering - Orders**: Add search (Name, Email, ID) and status filters to `AdminOrderList`.
- [ ] **Sorting & Pagination**: Add sortable columns and pagination to order lists.
- [ ] **Search & Filtering - Customers**: Apply similar enhancements to Customer list.

## Phase 2: Analytics & Insights
- [ ] **Dashboard Cards**: Implement Revenue, Top Meals, and detailed metrics.
- [ ] **Charts**: Add visual trends (Recharts or similar).
- [ ] **Export**: Add CSV export functionality for Orders.

## Phase 3: Workflow Automation
- [ ] **Status Automation**: Auto-update status based on triggers.
- [ ] **Email Triggers**: Visual editor or settings for email notifications.

## Phase 4: Advanced Features
- [ ] **Kitchen Management**: Production schedules and prep lists.
- [ ] **Delivery Routing**: Driver assignment and route optimization.
- [ ] **Inventory**: Ingredient level tracking.
